4 Motors: Electromagnetic actuator

  • generate force between e current + magnetic field
  • Lorentz Force: F = C x B
  • Force –> Motion
  • roter: i (wire loop) stater: q (electromagnetic)
  • DC motor: T proportional to i. T=Torque, i=current
  • Pe = VI = Pm = τ ω
  • Real motor: efficiency, n = Pm/ P2 [50%, 90%]
  • Stall Torque vs Free Speed, ideally straight line between
  • Purpose of gear is ensure operating highest efficiency and also desired torque
  • e-braking: ω is constrained when the two terminals are shorted
  • Ton / T : duty cycle
  • Vcycle = Ton /T Vin
  • To prevent reverse motor current, use a flyback diode
